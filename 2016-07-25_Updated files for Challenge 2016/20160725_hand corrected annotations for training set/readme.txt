Segmentation annotations with hand-corrected for PhysioNet/CinC Challenge 2016.
Four states were annotated in the file: S1, systole, S2 and diastole. 

each file has two columns: 

***************************************
column 1: start point  of each state
column 2: state name (S1, systole, S2 and diastole in turn)

the noisy signal segment is annotated as: '(N' for start point and 'N)' for end point
Sample rate: 2,000 Hz
