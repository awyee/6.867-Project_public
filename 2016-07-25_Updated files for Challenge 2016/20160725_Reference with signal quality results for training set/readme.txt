New classification and signal quality annotations for PhysioNet/CinC Challenge 2016.

Each training database file has three columns: 
***************************************
column 1: recording name
column 2: classification reference (1 for abnormal and -1 for normal)
column 3: signal quality reference (0 for unsure and 1 for heart sound recordings could be used for normal/abnormal classification)
